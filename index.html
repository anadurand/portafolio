Mis Proyectos

Aquí podrás encontrar algunos de los más importantes proyectos que realicé
durante el bootcamp (6 meses intensivos) en Laboratoria

POKEDEX


Del conocido anime Pokemón y del juego de Pokemon Go, este proyecto es una réplica de la Pokedex. En esta Pokedex podrás encontrar información de cada uno de los pokemón, su descripción, altura, peso, tipo, y debilidad.

Este proyecto fue realizado utilizando Bootstrap, HTML, JS componentes, CSS, AJAX, integrar con API de Pokeapi. El diseño de este proyecto se puede ver tanto en PC como dispositivos móviles.

Ver GitHub

LabCar


LabCar es una empresa de taxis que te permite pedir un taxi cuando quieras y donde quieras. Tiene una gran cantidad de conductores por todo el departamento(para este proyecto es Lima). Se ha comprobado que LabCar tiene las tarifas más bajas del mercado.

Este proyecto fue realizado utilizando Bootstrap, HTML, JS, CSS e integración con el API de Google Maps. Este proyecto tiene un diseño responsive, ya que se va utilizar más en dispositivos móviles que en desktop.

Ver Demo

Ver GitHub

YAPE


Este proyecto es la réplica de la aplicación del Banco de Crédito del Perú. Esta APP permite la transferencia de dinero desde un dispositivo móvil, sin necesidad de digitar el número de cuenta al cual depositar ni ingresar la clave del token.

Este proyecto fue realizado utilizando Materialize, HTML, JS componentes, CSS, integrar con API. Este proyecto tiene diseño responsive, su uso es para dispositivos móviles.
