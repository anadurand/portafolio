Información del proyecto

YAPE


Información del proyecto

Este proyecto es la réplica de la aplicación del Banco de Crédito del Perú. Esta APP permite la transferencia de dinero desde un dispositivo móvil, sin necesidad de digitar el número de cuenta al cual depositar ni ingresar la clave del token.

Este proyecto fue realizado utilizando Materialize, HTML, JS componentes, CSS, usar API dada por el profesor. Este proyecto tiene diseño responsive, su uso es para dispositivos móviles.
