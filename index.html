Mis Proyectos

LabCar


LabCar es una empresa de taxis que te permite pedir un taxi cuando quieras y en donde quieras. Tiene una gran cantidad de conductores por todo el departamento. Se ha comprobado que LabCar tiene las tarifas más bajas del mercado.

Este proyecto fue realizado utilizando Bootstrap, API Google Maps, HTML, JS, CSS. Este proyecto tiene diseño responsive, ya que se va utilizar más en dispositivos moviles que desktop.

YAPE


Este proyecto es la réplica de la aplicación del Banco de Crédito del Perú. Esta APP permite la transferencia de dinero desde un dispositivo móvil, sin necesidad de digitar el número de cuenta al cual depositar ni ingresar la clave del token.

Este proyecto fue realizado utilizando Materialize, HTML, JS componentes, CSS, usar API dada por el profesor. Este proyecto tiene diseño responsive, su uso es para dispositivos móviles.
